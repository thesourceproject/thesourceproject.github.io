Installing Visual Studio Code and Git in Windows

23 Dec '20 | 20 min read

#miscellaneous #vscode #git #windows #beginner

Visual Studio Code is a powerful free open-source text editor by Microsoft. VS Code is available for Windows, Linux, and macOS. Use it to code in any programming language, without switching editors. Visual Studio Code has support for many languages, including Python, Java, C++, JavaScript, and more.

Git is a free and open-source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

VS Code has integrated source control and includes Git support in-the-box. VS Code will leverage your machine's Git installation, so you need to install Git first before you get these features.

Getting Started

Download VS Code from the official website.

After downloading it, follow the instructions in the setup to install VS Code on your computer.
Then download Git from the official website. At the time of writing this article, the latest version available is 2.29.2.

After downloading it, follow the instructions in the setup to install Git on your computer.
Make sure you select Use Visual Studio Code as Git's default editor when prompted.
Continue the rest of the steps with the default options checked.

And there you go. You have now installed VS Code and Git on your computer.
Now time to write your first piece of code.

Create a new folder on your computer, right-click on it and select Open with Code in the context menu.
Note - It is important to open a folder(not a file) in VS Code otherwise the source control functions will not work.
This folder will now act as a workspace for your project.

Now go to Source Control in the sidebar and initialize the local repository. Ignore the Publish to Github button, for now. We will cover that in the next tutorial.

Now go to Explorer and create a new file named myFirstProject.html.

Now type !(yes, just an exclamation mark) in the editor and VS Code will show you that its an Emmet abbreviation.
Press Enter and the file will be populated by some boilerplate code automatically. Pretty cool, huh?

Now paste this code in the <body> tag of your document. 

<h1 style="margin-top: 2rem;color: #ff2f2f;font-family: sans-serif;text-align: center;">My First Project</h1>
<h2 style="color: #002e46;font-family: monospace;text-align: center;">For More Tutorials Visit <a href="https://thesourceproject.github.io" target="_blank">TheSourceProject</a></h2>

Run this HTML file in your browser to see the result.

Not bad for a start. Now go back to Source Control and type in first commit in the text box and press Ctrl + Enter to make your first commit in the repository.

A prompt will appear asking Would you like to stage all your changes and commit them directly?
Click Always. You can learn more about changes here

Oops! We get an error message because we haven't yet configured our Git user's name and email. Simply open the in-built Windows Powershell in VS Code(Ctrl + `) and type in the following two commands.
Don't forget to replace the name and email with your own. 

git config --global user.name "My Name"
git config --global user.email "myname@example.com"

Once you have configured the username and email try to make the commit once more and it should work as normal.

And there you go! You have created your first local Git repository and made your first commit. You're now ready to start creating! You may see our next tutorial where we show you how to publish this project to Github.
