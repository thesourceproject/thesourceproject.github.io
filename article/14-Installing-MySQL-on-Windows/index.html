Installing MySQL on Windows

11 Jun '21 | 5 min read

#backend #database #sql #mysql #windows #beginner

If you are just starting with backend development or want to dabble in SQL, you will need to understand databases. In this tutorial, we will learn how to install the world's most popular database on Windows.

Getting Started

MySQL also comes with an installer that can simplify this process though I prefer the ZIP archive method as it offers more flexibility and can easily be upgraded or removed.

Download MySQL Community Server from the official website. Look for Windows (x86, 64-bit), ZIP Archive under Other Downloads.
At the time of writing this article, the latest version available is 8.0.25.

Extract this ZIP archive and create an option file named my.ini in it. Then, create an empty directory data anywhere on your system.

Now, open the my.ini file using notepad or any other text editor and paste the following. 

[mysqld]
# set basedir to your installation path
basedir=E:/mysql
# set datadir to the location of your data directory
datadir=E:/mydata/data

Don't forget to replace the above paths with the actual ones. Here, basedir refers to the location where you extracted the ZIP archive.
Note - You can shorten the name of your extracted folder to just mysql as in the example. It will make it easier to access it using the command line.

Now, open the bin folder inside the mysql folder in Windows PowerShell. You can do this by pressing and holding shift and right-clicking the folder. In the context menu, it will give you an option saying Open PowerShell window here.

Enter the following two commands in the PowerShell window. 

./mysqld.exe --initialize-insecure
./mysqld.exe

The first command will initialize the data directory without a root password. The second command will start the MySQL server. Now, let's access our MySQL installation using the command-line client.

Leave this window running and open another PowerShell window in the same directory. Then, run the following command. Notice this time it's mysql and not mysqld. 

./mysql.exe -u root

You will log in to MySQL as the root user. Before creating any new databases, we should set a password for the root user.

Enter the following command in MySQL. Replace root-password with any password of your choice. 

ALTER USER 'root'@'localhost' IDENTIFIED BY 'root-password';

Now, exit MySQL by typing exit and login again using the new password. 

./mysql.exe -u root -p

You will be prompted for the root password. We are now ready to play around with SQL.
Type the following SQL queries in command line. 

CREATE DATABASE classroom;
USE classroom;
CREATE TABLE student (id INT NOT NULL AUTO_INCREMENT PRIMARY KEY, name VARCHAR(20), marks TINYINT UNSIGNED);
INSERT INTO student (name, marks) VALUES ('Ryan Smith', 25), ('James Adams', 33), ('Jeremy Williams', 29);

Now, we have created a database "classroom" and a table "student".

Let's see the contents of the student table. Enter these SQL queries. 

SELECT * FROM student;
DELETE FROM student WHERE name = "James Adams";
SELECT * FROM student;

We deleted a row using the DELETE query. Now, our table looks like this.

And there you go! Our MySQL installation works like a charm. It's also easy to upgrade to a newer version later. Just extract the new ZIP archive and create an option file like this one while referring to the same data directory.

We have installed MySQL in Windows in the simplest way possible. Now, why not play around with some SQL queries on your own?
