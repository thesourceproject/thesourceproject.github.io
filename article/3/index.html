Python fundamentals

6 Jan '21 | 15 min read

#basics #python #beginner

Python is an interpreted, high-level and general-purpose programming language. Python is being used in web development, machine learning, and even desktop applications. Its easy-to-learn syntax is very well suited for beginners as well as experienced programmers.

Getting Started

First, we need to install Python. You can download Python from the official website.
At the time of writing this article, the latest version available is 3.9.1.
Once the download is complete, you may begin the installation process.
Note - Make sure the Add Python 3.9 to PATH option is checked. This will allow you to access the Python interpreter in the Windows Powershell from anywhere.

Open a Powershell window, type in python, and press Enter.
This will open the Python interpreter. You can try out Python here.

Let's start with everyone's favorite, the Hello World! program. Enter this code in the python shell and press Enter. 

print('Hello World!')

It should print Hello World!(Obviously). But now let's try some statements. Enter these statements one by one in the python shell. 

x = 5
y = 7
# Can also be written as --> x, y = 5, 7
z = x * y
print(z)

It should print 35. What we did here was that we first declared a variable x while giving it a value of 5. Similarly, we also declared y and gave it a value of 7. We then stored the product of x and y in z(In Python, as in most programming languages, the * symbol means multiplication).
After that, we simply print the value of z(i.e. 35).
Note - You can add comments by writing # and then typing in the desired comment. These do not change the output of the program.

Now let's take it up a notch. Till now we were using the Python shell to type and run our code. It's a great tool to try out Python but real-life applications are created in Python scripts. So now we will learn how to use the popular text editor Visual Studio Code to create Python scripts.
Note - For instructions on how to install and work with VS Code, refer to this tutorial.

Create a new folder on your computer named TheSourceProject, right-click on it and open it in VS Code.
Now create a new file myFirstProgram.py and paste in the following code. 

# Formula for simple interest --> (p * r * t) / 100
p1, p2, p3 = 1200, 1500, 2000
r1, r2, r3 = 5, 10, 8
t1, t2, t3 = 5, 2, 3
a1 = (p1 * r1 * t1) / 100
a2 = (p2 * r2 * t2) / 100
a3 = (p3 * r3 * t3) / 100
print(a1, a2, a3, sep=', ')

Save the file and then open the in-built Powershell(Ctrl + `) and it should show you the current working directory. Now type in the following command. 

python myFirstProgram.py

This command will run your program and you will get the output as three comma-separated numbers 300.0, 300.0, 480.0(The default separator for the print function is a single space but it can be changed to anything as in the example).
Note - It is important to know that variable/function names can only start with either a letter or an underscore and may contain digits, letters and underscores after that.

You might have noticed that we are reusing the same logic for calculating simple interest for 3 different values which is causing duplication in our code. Functions(Or methods) will help us avoid this duplication.
Erase everything and type in the following statements and run the program. Don't forget the whitespace before the return statement. 

# Same program as above, now using a function
def simpleInterest(p, r, t):
  return (p * r * t) / 100
p1, p2, p3 = 1200, 1500, 2000
r1, r2, r3 = 5, 10, 8
t1, t2, t3 = 5, 2, 3
a1 = simpleInterest(p1, r1, t1)
a2 = simpleInterest(p2, r2, t2)
a3 = simpleInterest(p3, r3, t3)
print(a1, a2, a3, sep=', ')

Now we can reuse our simpleInterest function as many times as needed. Also if we have to modify the logic we only need to do it in one place. Therefore, this code is a lot more maintainable than the previous one.
Note - Notice that it is necessary to add white-space when starting a new block of code, like after declaring a function, loop, or an if-else statement. Using camel-case for variable/function names is optional but usually preferred.

In our previous code, we ran our simpleInterest function 3 times, what if wanted to run it 100 times or even 1000 times. Now we need to learn about loops. Run the following program. 

for i in range(5):
  print(i)

This is the simplest way to run a for loop. Here an iterator i is created and its value changes from 0 to 4(i.e less than 5). The print(i) statement in the block prints the current value of i each time the loop is executed(In our case 0, 1, 2, 3, 4). We can also modify the for loop so that it iterates through our required range. 

for i in range(10, 101, 4):
  print(i)

Here our iterator i is created and its value changes from 10 to 100(i.e less than 101) with a step value of 4(i.e. 10, 14, 18, 22...).
