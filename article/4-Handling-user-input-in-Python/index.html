Handling user input in Python

8 Jan '21 | 5 min read

#basics #python #beginner

While developing Python applications we often need to allow user interaction, either to obtain necessary data or to direct the flow of our program. Python enables this functionality through its inbuilt input() function. In this tutorial, we will learn how to use it to create dynamic applications.
Note - This tutorial assumes you have some basic understanding of Python. You may refer to our previous tutorial before you continue.

Getting Started

Let's start with a basic example.

name = input('Enter your name: ')
print('Hello', name)

When you run this, Python will prompt you to enter your name in the terminal. Type anything and press enter to continue. It will print Hello, then space, and then your name.
What we did here was that we captured the user input and stored it in a variable called name and then printed a message using that variable. The input() function can also take an optional string that can be shown to the user before giving an input which in our case was Enter your name:

Now let's perform some mathematical operations on user input but before we continue we need to learn a bit about data types in Python. Python has some basic data types -

  • Integer: represents mathematical integers.
  • Floating-Point Number: numbers with a fractional part.
  • Boolean: a data type that can only have one of the two values, True or False.
  • String: a set of characters such as letters, numbers, symbols, etc.

The input() function returns a string that represents the captured user input but we cannot directly perform mathematical operations on strings in Python.
Therefore, we need to convert our input string to an appropriate data type first.
Run the following. 

number = int(input())
print('Cube of the given number is ', number ** 3)

Before storing our user input in the variable number, we convert it to an integer using the int() function. Now we use the **(Exponentiation operator) to find the cube of the integer.
Note - if you supply a non-integer value to the program, you will receive a message like ValueError: invalid literal for int() with base 10. This is because the int() function expects an integer string to convert its type.

Similarly, we can input floating-point numbers by using the float() function. 

num1 = float(input())
num2 = float(input())
print('Sum of the given numbers is ', num1 + num2)

This program can not only take integers(e.g. 1, 4, 200512) but also fractional-numbers(e.g. 3.14, 5.6, 925.163) as input.
Like in the previous program, it will throw a ValueError if a non-numeric value is supplied.

We can also use the input() function inside a loop. This is useful when we do not know in advance how many times the user input is needed. 

total = 0
numbers = int(input('How many numbers do you want to enter: '))
for i in range(numbers):
  total = total + float(input()) # total += float(input())
print('Sum of the given numbers is ', total)

First, we create a variable total(not sum, as sum is an in-built function of Python and therefore, should be avoided). Then we ask the user how many numbers he/she will input.
After that, we run a for loop that many times, each time asking for a number and adding it to our total.

What if the user provides all input values in a single line separated by spaces? For example, 10 25 12 47. How will we unpack the individual values?
We know that the input() function returns a string. Python strings have an inbuilt function called split() which splits the string based on a given delimiter(by default, whitespace) and returns a list containing the individual strings. 

fruits = input('Enter the names of fruits separated by comma: ')  # Apple, Banana, Cherry
print(fruits.split(', '))  # ['Apple', 'Banana', 'Cherry'] (Split the string using the delimiter ', ')

Notice that the split function will return a list of strings. But, what if we had numbers in our input? We will simply loop over each string in this list and call the int() or float() function upon it. 

numbers = input('Enter the numbers separated by space: ')  # 10 25 12 47
numberList = numbers.split()  # By default, the string will be split based on whitespace.
print(numberList)  # ['10', '25', '12', '47']
for i in range(len(numberList)):
    numberList[i] = int(numberList[i])  # Convert each string to an integer.
print(numberList)  # [10, 25, 12, 47]

Luckily, Python has an easier way of doing this with the help of the map() function. The map() function returns an iterator that applies the given function to each of the items in the iterable(list). We wrap this in a list() function call so that we obtain the list from this iterator. 

numbers = input('Enter the numbers separated by space: ')  # 10 25 12 47
numberList = list(map(int, numbers.split()))
print(numberList)  # [10, 25, 12, 47]

That's all you need to know about handling user input in Python. Now you can practice making some programs requiring user input on your own.
