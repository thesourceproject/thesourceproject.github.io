Creating a countdown timer using Javascript

9 Feb'21 | 20 min read

#frontend #html #css #javascript #beginner

In this tutorial, we will learn how to create a simple countdown timer application using Javascript that can then be run in your web browser.

Getting Started

First, we need to write the HTML for our countdown timer.
Create an empty file on your computer named myCountdownTimer.html and paste in the following markup in the body tag. 

<div id="countdownTimer">
  <input id="duration" type="number" placeholder="Timer Duration" />
  <fieldset id="controls">
    <button id="startButton" title="Start">Start</button>
    <button id="stopButton" title="Stop">Stop</button>
    <button id="resetButton" title="Reset">Reset</button>
  </fieldset>
  <p id="timer">0.0</p>
</div>

We are giving each of our elements a unique id attribute. This will help us with the styling and also allow us to access the element with Javascript.
Note - The class attribute should be used for styling elements but since all of our elements are unique, we may use id. Also, the id will be required for our Javascript.

If you open this HTML file right now, you'll notice that it doesn't look anything like the image above. This is because we haven't added any CSS yet. Time to add some styling.
Paste the following CSS inside a style tag in the head section of your HTML file. 

@import url("https://fonts.googleapis.com/css2?family=Open+Sans&display=swap");

First, we import Open Sans font-family from Google Fonts. Google Fonts has a large catalog of free fonts that you may use in any of your projects. 

body {
  margin: 0;
  display: flex;
  min-height: 100vh;
}

Then we add some styling to the body tag. We set its min-height property to 100vh(vh means hundredths of the viewport height). This makes the body at least as tall as the height of the current viewport. This, along with the display: flex property, will help us to center the countdown timer on the screen. 

#countdownTimer {
  margin: auto;
}

We use margin: auto property on the countDownTimer element to equally apply margin on all 4 sides, this will center the element horizontally as well as vertically.
Note - We referred to the body tag with body but the countDownTimer element with #countDownTimer(with a #). The # symbol is a CSS selector used to apply styles on a specific id. If we want to style any HTML elements, we can do it so using their names(such as h1, input), without any CSS selectors. 

input,
button,
p {
  font-family: "Open Sans", sans-serif;
  padding: 0.5rem;
  font-size: 1.25rem;
  border: none;
  text-align: center;
}

We add padding of 0.5rem(1rem equals the font size of the html element - which for most browsers has a default value of 16px) to our elements and increase the font-size to 1.25rem. We continue to add the rest of the styles, including our imported font-family.
Note - We applied the font-family property to the input, button, and p elements instead of just applying it on the body tag. This is because form elements such as input, button, textarea, and select don't inherit the font-family automatically. 

button {
  border-radius: 0.25rem;
  cursor: pointer;
}

Now we add a border-radius of 0.25rem to the button element to make its corners slightly round. The cursor: pointer property changes the mouse cursor to a pointer when it's over this element. 

#duration {
  outline: none;
  border-bottom: 2px solid lightgrey;
}

#duration:focus {
  border-color: darkblue;
}

outline: none removes the black outline which is shown when the element is focused upon. Instead, we will add a light-grey bottom border to the element which will change to dark-blue on focus. We achieve this using the :focus pseudo-class.
Note - border-bottom is a shorthand property for setting border width, style, and color. 

#controls {
  margin-top: 0.5rem;
  padding: 0;
  border: none;
  display: flex;
  justify-content: center;
}

We use the display: flex and justify-content: center properties to center the buttons.
Note - You can learn more about the flex display property in this tutorial. 

#startButton {
  background-color: lightgreen;
}

#stopButton {
  margin-left: 0.5rem;
  margin-right: 0.5rem;
  background-color: orangered;
  color: white;
}

#resetButton {
  background-color: yellow;
}

#timer {
  margin: 0;
  font-size: 10rem;
}

We go on to add some background color and text color to the buttons and the timer element.

It looks much better now but we haven't added any functionality yet. So let's move on to Javascript. 

const duration = document.getElementById("duration");
const startButton = document.getElementById("startButton");
const stopButton = document.getElementById("stopButton");
const resetButton = document.getElementById("resetButton");
const timer = document.getElementById("timer");

let interval;

First, we fetch all of our necessary elements by using the document.getElementById() function. We use the same ids that we used in the HTML.
Note - We used const for declaring the variables for the HTML elements because they will never be reassigned whereas the interval variable(declared using let) will be reassigned later.

Now that our elements are ready, we will implement the functionality of the start button using the addEventListener function. 

startButton.addEventListener("click", () => {
  let remainingTime = Number(duration.value);
  timer.innerText = remainingTime;
  interval = setInterval(() => {
    remainingTime = remainingTime - 0.1;
    timer.innerText = remainingTime;
    if (remainingTime == 0) {
      clearInterval(interval);
    }
  }, 100);
});

What we did here was that created a temporary variable remainingTime that will store the remaining time which will be shown to the user. Then we use the setInterval() function to create a routine that will be run at the end of every 100 milliseconds.
Note - The setInterval() function returns an Interval ID which we store in the interval variable. This will be used later to stop the routine.

In our routine, we subtract 0.1(100 milliseconds) from remainingTime and update the timer. Then we check if remainingTime is 0 and pass our interval variable to the clearInterval() function to stop the routine.
Now let's try to run this. When you click the start button the timer starts normally but soon you will see something like this.

This is because of floating-point arithmetic. Due to their binary nature, some decimal numbers cannot be represented with perfect accuracy in Javascript. Therefore we will use the toFixed() function to truncate the remaining time to 1 decimal place. Change remainingTime = remainingTime - 0.1 to this. 

remainingTime = (remainingTime - 0.1).toFixed(1);

Now if you run the timer again, it will run normally. But there's one more problem.
Try to click the start button again before the timer runs out. You'll notice that you have started 2 timers simultaneously and the remaining time keeps switching back and forth between the two.
We can solve this problem simply by clearing the interval before we start the timer. We'll also add additional checks to ensure that our timer does not accept invalid durations. Now our function looks like this. 

startButton.addEventListener("click", () => {
  clearInterval(interval);
  let remainingTime = Number(duration.value);
  if (Number.isInteger(remainingTime) && remainingTime > 0) {
    timer.innerText = remainingTime;
    interval = setInterval(() => {
      remainingTime = (remainingTime - 0.1).toFixed(1);
      timer.innerText = remainingTime;
      if (remainingTime == 0) {
        clearInterval(interval);
      }
    }, 100);
  } else {
    alert("Please enter a valid duration!");
  }
});

Now what's left is the easy stuff. We need to add the functionalities for the stop and reset buttons. 

stopButton.addEventListener("click", () => {
  clearInterval(interval);
});

resetButton.addEventListener("click", () => {
  clearInterval(interval);
  timer.innerText = "0.0";
});

And there you go. We have successfully created a countdown timer app using Javascript. Now try building a stopwatch application on your own with what you've learned and keep learning.
