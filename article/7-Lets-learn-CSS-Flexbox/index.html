Let's learn CSS Flexbox

2 Mar '21 | 15 min read

#frontend #html #css #intermediate

If you're just starting with HTML and CSS, you must have heard about Flexbox. This tutorial will teach you the basics of Flexbox so you can use it in your projects in no time.

But what is Flexbox and why should we use it?

Before Flexbox, positioning of items inside its parent container was very cumbersome and in some cases impossible. It required the use of float and position properties and the advent of mobile devices with different screen sizes didn't make it any easier.

Flexbox solves these problems by providing an easy method to arrange items in one dimension inside a container.

Getting Started

To demonstrate the power of Flexbox we will create a parent container and place a few items in it. 

<div class="parent">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  ...
</div>

And we will style the child items so that we can distinguish them from the parent and other items. 

.parent > div {
  border: red dashed 2px;
  border-radius: 2rem;
  padding: 2rem;
  font-size: 2rem;
  text-align: center;
}

Now we will learn about different properties commonly used to work with Flexbox using our interactive examples.

To use Flexbox we need to use the flex display property.
Flex items are initially arranged in the row direction(left to right). This can be changed using the flex-direction property.

The flex-wrap property will allow us to control whether the items are fitted in a single line(and possibly overflowing) or wrap across multiple lines as needed.
wrap-reverse will wrap the items from bottom to top.

The justify-content property controls the position of the child elements in the main axis(direction of flex items).
This property is useful for distributing the remaining space left around child items.

The align-items property controls the positioning of flex items on the cross axis(column direction if the flex items are arranged row-wise and vice-versa).
baseline arranges the items so that their baselines are aligned.

The align-content property is similar to the justify-content property but this works along the cross axis.
This property only works when the flex items are wrapped onto multiple lines. This means that the flex-wrap property must be set to either wrap or wrap-reverse.

The flex-grow property indicates how much the flex item grows compared to the other flex items along the main axis.
Its initial value is 0 meaning that the flex item will not grow to take over the remaining space.

The align-self property is similar to the align-items property but for individual items.
Use this property on child items to override the parent container's align-items property.

We've covered the most commonly used Flexbox features but there are more(e.g. flex-flow, flex-shrink, flex-basis, etc) that are not used as often. You can learn more about them here.
